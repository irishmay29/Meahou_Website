# AkamaiPOS Website Presentation Outline

## 1. Website Snapshot
- **Website name:** AkamaiPOS
- **Purpose:** Promote POS, payments, hardware, software, and IT support services for Hawaii-based and distributed businesses.
- **Primary audience:** Retail, restaurant, hospitality, healthcare, wholesale, and professional service operators.
- **Core value proposition:** "Cost Saving Revenue Growth Systems" with expert support and custom solutions.

## 2. Presentation Goal
- Explain what the site does.
- Show how each page supports lead generation.
- Highlight brand positioning (local trust + modern technology + full-service support).

## 3. Site Structure (Sitemap)
- **Home** (`/`)
- **POS Hardware** (`/hardware`)
- **Suppliers** (`/suppliers`)
- **Contact Us** (`/contact`)

## 4. Shared Experience Across All Pages
- Sticky header with brand, navigation, and **Book a Demo** CTA.
- Theme toggle (light/dark mode persistence).
- Animated ambient background and reveal effects.
- Footer with support link.
- Live support widget integration (Tawk chat).

## 5. Page-by-Page Outline

### Home Page
**Role in funnel:** Awareness + trust + conversion.

Key sections to present:
- Hero statement: POS system built for Hawaii and beyond.
- Service ticker: quick scan of offerings (POS, payments, gift cards, mobile ordering, inventory, time clock, etc.).
- Business impact section: revenue/cost-savings analytics visual and outcomes.
- Experience section: 20+ years and multi-industry expertise.
- Coverage section: Hawaii islands, Guam, mainland, and beyond.
- Third-party system support section: works with major legacy and current brands.
- Custom programming section: tailored software development capability.
- IT and cloud support section: full infrastructure and backup services.
- Services grid: complete portfolio in one place.
- Support CTA bands: 24/7 support and direct phone call actions.

Primary CTA options:
- Start Your Rollout
- Talk to an Expert
- Get a Free Consultation
- Call (808) 843-8000

### POS Hardware Page
**Role in funnel:** Consideration + product fit.

Key sections to present:
- Page intro explaining editable hardware catalog model.
- Dynamic sample hardware sections from data source.
- Product-level detail blocks (category, SKU, includes list, description).
- Closing conversion CTA to request pricing.

Primary CTA:
- Ask About Hardware Pricing

### Suppliers Page
**Role in funnel:** Credibility + ecosystem confidence.

Key sections to present:
- Intro positioning trusted brand partnerships.
- Logo grid of supplier/manufacturer partners.
- Coverage + direct contact block.

Primary CTA:
- Become a Partner

### Contact Page
**Role in funnel:** Conversion.

Key sections to present:
- Intro message for business inquiry.
- Direct call information.
- Structured contact form capturing:
  - Name
  - Company
  - Business type
  - Referral source
  - Phone
  - Message
- Submission flow opens prefilled email to support.

Primary CTA:
- Contact Our Team

## 6. Design and Brand Language
- Island-inspired identity with modern SaaS visual treatment.
- Typography mix: strong headings + readable body copy.
- Color strategy: tropical teal/coral accents with professional contrast.
- Visual motifs: glass cards, gradients, icons, map/coverage cues, analytics-style visuals.

## 7. Technical Notes (High-Level)
- Frontend: React + Vite.
- Routing: React Router with 4 public pages.
- Interaction patterns: animated reveals, responsive navigation, persistent theme preference.
- Backend foundation exists in project (`server/`) for future integrations.

## 8. Suggested 3-5 Minute Talk Track
1. **Opening (30s):** What AkamaiPOS is and who it serves.
2. **Site map (30s):** Home, Hardware, Suppliers, Contact.
3. **Home page deep dive (90s):** Value proposition, services breadth, trust signals, support.
4. **Hardware + Suppliers (60s):** Product capability and partner ecosystem credibility.
5. **Contact conversion flow (30s):** How leads are captured.
6. **Close (30s):** Why this site works (clear messaging + trust + direct CTAs).

## 9. One-Slide Executive Summary
"AkamaiPOS.com is a conversion-focused service website that combines local credibility, broad POS/IT capability, and direct support-driven calls to action across four core pages: Home, Hardware, Suppliers, and Contact."
